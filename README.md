# Subway_charge_optimazation
<div align="center">
<h2>[2023] 혼잡도를 바탕으로 한 지하철 요금 최적화 프로젝트</h2>
2021.1.1~2022.12.21.까지의 지하철 혼잡도를 구하고 Cov1D-GRU Autoencoder로 학습을 하여 2022.12.22.~12.31까지의 지하철 혼잡도를 추론하였다.추론된 지하철 혼잡도를 바탕으로 혼잡도에 비례하는 Dynamic Pricing을 구현하고 최적의 지하철 요금을 도출하였다. 이를 Test셋으로 백 테스트하였고 Streamlit을 활용하여 대시보드 형태로 시각화하였다.
</div>

## 개요
- 프로젝트 이름: 혼잡도를 바탕으로 한 지하철 요금 최적화 프로젝트 🚇
- 프로젝트 지속기간: 2023.05.31-2023.06.31(4주)
- 개발 엔진 및 언어: Python
- 멤버: 임도현(본인)

## 배경
- 지하철 운영 적자의 누적
  -- 2022.10월 기준 서울교통공사의 지하철 운영적자 누적액은 17조 5,942억원
  -- 수송원가에도 못 미치는 낮은 운임이 적자의 가장 큰 원인
- 출퇴근 시간대 지옥철로 불리는 특정 호선의 혼잡도 문제
  -- 지하철 혼잡도가 높으면 육체적 정신적 피로도 상승, 사회적 쾌적성 저하 비용 발생
  -- 출퇴근 시간대 몰리는 인파로 안전사고위험이 항상 있음
- 현 지하철 요금제도는 균일요금제로 수요에 대응하기 어려움
  -- 첨두시간대 교통혼잡을 해결하기 어려워 대중교통 서비스의 질 하락의 원인
  -- 시간대 별로 변하는 수요에 부응하지 못해 수요관리 측면에서 취약함
- 해외의 경우 대중교통 변동요금제를 도입
  -- 도쿄, 뉴욕, 런던 등에서는 첨두시와 비첨두시에 따라 대중교통부문에 변동요금제를 적용
  -- 비첨두시엔 대중교통요금을 10∼50%까지 할인하여 대중교통의 수요창출을 유도
